# aura-house-app

智能家居App - 高保真原型设计

## 项目说明

这是一个智能家居App的高保真原型设计，使用HTML + Tailwind CSS实现。

## 主要文件

- `prototype-index.html` - 主预览页面，展示所有界面
- `home.html` - 首页
- `devices.html` - 设备管理
- `device-light.html` - 智能灯光控制
- `device-air.html` - 空气净化器控制
- `device-temp.html` - 温度控制
- `rooms.html` - 房间列表
- `scenes.html` - 场景管理
- `shop.html` - 商城
- `profile.html` - 个人中心
- `outdoor.html` - 户外天气

## 使用方法

### 本地运行

1. 启动本地服务器：
   ```bash
   # Windows
   双击 start-server.bat
   ```

2. 访问预览页面：
   ```
   http://127.0.0.1:8000/prototype-index.html
   ```

### 在线预览

访问部署的网站查看所有界面。

## 技术栈

- HTML5
- Tailwind CSS
- Font Awesome 6
- 响应式设计（iPhone 15 Pro尺寸）

## 设计特点

- iOS风格设计
- 毛玻璃效果（Glassmorphism）
- 现代化UI界面
- 高保真原型
